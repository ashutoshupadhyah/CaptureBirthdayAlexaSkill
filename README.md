# CaptureBirthdayAlexaSkill
Captures the {day} {month} {year} of your birthday. If not mentioned properly Alexa will ask again. 
index.js file is the skill code with captureBirthdayIntentHandler and HasBirthdayLaunchRequestHandler. 
Package.json file has "name:", "version:", "description:", "author:", "Licence:" and dependencies.
util.js file the bucket description and export function.
Read data is stored in AWS S3 bucket and accessed through AWS Lambda.
